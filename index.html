    ¿Cómo funciona este modelo?

    Proxy origen-destino: rutas mínimas con ruido sobre la red caminable real (calles + red peatonal de OSM). No es un ABM con colisiones: es un modelo de asignación reproducible (misma semilla → mismo resultado), respaldado por space syntax (Choice correlaciona r=0.7–0.9 con conteos peatonales reales). Muestra flujo simulado potencial, nunca conteos observados.
